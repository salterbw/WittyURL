  • PÜMA

    Work in Progress

    Each movie on IMDb has a ten-point scale that users can use to review (based on whatever criteria they feel like). PÜMA (Probably Über Movie Advice) seeks to predict this score using a combination of movie metadata acquired from IMDb and common machine learning techniques (such as Neural Networks or Na√≠ve Bayes).

    This project began as a class project Spring 2013 for CSCI 4707 (Databases) as well as CSCI 5521 (Pattern Recognition) at UMN. Extension of this work will be part of research I'm doing with Carl Sturtivant to develop machine learning techniques in D using concurrency wherever feasible. Additionally, work is being done as part of a class project for CSCI 5525 (Machine Learning) Fall 2013.

    Eventually I intend to develop a web-based front end, likely using Node.js and likely viewable as a subdomain here at saltsaur.us.

  • KIX to C++ Parser/Translator

    Completed

    KIX was an artificial, functional programming language similar to Scheme. As a class project for CSCI 3081 (Program Design & Development) I designed and built a language parser which could take a file in our faux language and convert it into C++, first by determining the Abstract Syntax Tree and then building the C++ file from this tree.

    The code for this project is unavailable as it is still being used as the project each semester.
